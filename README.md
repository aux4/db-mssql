# aux4-db-mssql

MSSQL (Microsoft SQL Server) database tools using the mssql client for the aux4 ecosystem.

This package provides a Microsoft SQL Server database driver that follows the aux4 db driver specification outlined in [db/package/README.md](../db/package/README.md).

- [aux4 hub](https://hub.aux4.io/r/public/packages/aux4/db-mssql)
- [package documentation](./package/README.md)

## Installation

```bash
aux4 aux4 pkger install aux4/db-mssql
```

## Development

```bash
# Install dependencies
npm install

# Build the executable
npm run build

# Run in development mode
npm run dev
```
